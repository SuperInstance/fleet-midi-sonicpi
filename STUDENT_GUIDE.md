# Student Guide...

## What does this repo do?

Generates Sonic Pi live_loop code from MIDI note data. Paste the output into Sonic Pi and hear the fleet play in real time.

## How it connects

**Pulse** is the live playback bridge. Send MIDI from any agent, get live audio.
